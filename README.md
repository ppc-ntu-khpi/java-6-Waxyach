# Генерація матриці Фібоначчі

Цей репозиторій містить виконання практичного завдання з обчислення та генерації двовимірної матриці, заснованої на правилах послідовності Фібоначчі.

## Опис завдання

Необхідно заповнити матрицю розміром `N x M` числами, кожне з яких дорівнює сумі двох попередніх елементів.

**Правила обчислення:**
* Базові елементи $a_{11}$ та $a_{12}$ дорівнюють **1**.
* Усі наступні елементи обчислюються за формулою: $a_{ij} = a_{ij-1} + a_{ij-2}$ (з урахуванням наскрізного переходу між рядками матриці, де послідовність не переривається в кінці рядка).

## Приклад роботи програми

Для вхідних параметрів `N = 3` (рядки) та `M = 3` (стовпці), програма згенерує таку матрицю:

| Стовпець 1 | Стовпець 2 | Стовпець 3 |
| :---: | :---: | :---: |
| **1** | **1** | **2** |
| **3** | **5** | **8** |
| **13** | **21** | **34** |
